Hvad kan du få hjælp til?

  • Installation og første opsætning (Mac, iPad, iPhone)
  • Forbindelse til server / backend-URL
  • Bygninger, belægning, personer og opgaver
  • Kamera, fotos, kalender og AI-notetager
  • Fejl, nedbrud eller uventet adfærd
  • Spørgsmål om privatliv og data

Hurtige tips

  • Server: Angiv den korrekte backend-URL under indstillinger (fx intern server på jeres netværk).
  • Samme Wi‑Fi: Enheder skal typisk være på samme netværk som serveren ved lokal drift.
  • Tilladelser: Kamera, fotos, mikrofon og kalender kan aktiveres under iOS/macOS Indstillinger → Indkvartering.
  • Demo-data: Brug kun testdata i pilot/demo. Undgå rigtige CPR-numre på åbne testservere.

Når du skriver

Det hjælper, hvis du medsender:

  • App-version (fx 1.0.0)
  • Enhed og system (fx iPad, iOS 18 / Mac)
  • Kort beskrivelse af problemet og hvad du forventede
  • Skærmbillede (uden personfølsomme oplysninger, hvis muligt)
